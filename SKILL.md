# 口咽癌根治性（非手术）放疗靶区勾画

> **自包含模块** —— 口咽癌未手术、行根治性放疗/放化疗时加载。
> 若患者有诱导化疗史，参见第三节（GTV 退缩后处理）。
> 若为术后 PORT，改用 `port-oropharynx-postop`。
>
> 朱国培 · 上海交通大学医学院附属第九人民医院口腔颌面-头颈肿瘤科

## 根治性 RT 与术后 PORT 的根本差异

| | 术后 PORT | 根治性 RT |
|---|---|---|
| **靶区基础** | 手术床 + 病理报告 | **GTV（影像可见肿瘤）** |
| **核心文件** | 手术记录 + 病理 | **诊断 MRI + PET-CT** |
| **剂量方案** | 序贯（60/54Gy + 缩野加量），**不用 SIB** | **SIB（同步推量）** |
| **关键风险** | 漏照（手术床边界不清） | **GTV 勾画不足**（尤其诱导化疗后） |
| **HPV 影响** | 影响预后判断 | 直接影响 **是否降剂量** |
| **解剖** | 术后变形 | 正常解剖（除非诱导化疗后退缩） |

## 一、GTV 勾画——根治性 RT 的核心

### 1.1 GTV 定义

| 体积 | 定义 | 影像依据 |
|------|------|---------|
| **GTV-P（原发灶）** | 影像可见的原发肿瘤 | 诊断 MRI（T1+C 增强）为主；PET-CT 辅助判断边界 |
| **GTV-N（淋巴结）** | 影像可见的转移淋巴结 | MRI + PET-CT；坏死/囊变区全部纳入 |

### 1.2 CTV 外放（从 GTV 出发——不是从手术床出发）

| CTV 层级 | 外放 | 内容 |
|---------|------|------|
| **CTV-HR（高危）** | GTV-P + 5-10mm | GTV 周围亚临床侵犯区。骨/筋膜屏障处修回 |
| **CTV-IR（中危）** | CTV-HR + 5-10mm | 邻近解剖亚区的预防覆盖 |
| **CTV-LR（低危/预防）** | 选择性淋巴结区域 | 见第四节颈部管理 |

> **与 PORT 的本质区别**：PORT 的 CTV 是"手术床复合体积"；根治性 RT 的 CTV 是"GTV + 逐层外放"。

### 1.3 亚部位特殊要点

| 亚部位 | GTV 勾画注意 |
|--------|------------|
| **扁桃体** | 注意腭舌弓/腭咽弓的前后扩展；咽旁间隙侵犯需 MRI 冠状位判断 |
| **舌根** | 舌会厌皱襞、会厌谷、会厌前间隙——MRI 矢状位关键 |
| **软腭** | 口腔面+鼻咽面双重评估；咽鼓管圆枕受累→需覆盖至鼻咽 |
| **咽侧壁** | 区分口咽 vs 下咽——后界=咽后壁黏膜面，不是椎前筋膜 |

## 二、SIB 剂量方案（根治性 RT 的核心技术）

> 九院实践：**根治性 RT 使用 SIB（同步推量）**——与术后 PORT 不用 SIB 形成鲜明对比。

### 2.1 标准 SIB 方案

| PTV 层级 | 处方 | 单次剂量 | 适应证 |
|---------|------|:--:|------|
| **PTV-HR（高剂量）** | 70 Gy / 35fx | 2.0 Gy | GTV-P + GTV-N |
| **PTV-IR（中剂量）** | 63 Gy / 35fx | 1.8 Gy | CTV-HR（亚临床高危区） |
| **PTV-LR（低剂量）** | 56 Gy / 35fx | 1.6 Gy | 选择性淋巴结预防区 |

### 2.2 关键剂量学原则

| 原则 | 说明 |
|------|------|
| **95% 处方剂量覆盖 ≥95% PTV** | D98% ≥ 95% 处方剂量 |
| **热点 ≤107%** | D2% ≤ 107% 处方剂量；热点必须在 PTV 内 |
| **GTV 内无冷点** | GTV D98% ≥ 100% 处方剂量——GTV 内不可接受冷点 |
| **PTV 边缘冷点** | 体表/空腔表面的 PTV 边缘冷点（<95%）可接受——注明位置 |

### 2.3 HPV+ 降剂量——正在演变的领域

> ⚠️ **未写入指南、仍在临床试验阶段。** 以下为前沿方向，非标准实践。

| 试验/方向 | 方案 | 状态 |
|----------|------|------|
| NRG-HN005 | 60 Gy / 30fx（降剂量臂） vs 70 Gy | Phase III 进行中 |
| 降剂量至 60 Gy 思路 | p16+、<T3、<N2b、<10包年吸烟 | 仅限临床研究 |
| 诱导化疗后 CR→降剂量 | GTV 按退缩后勾画→CTV外放不变→剂量60-66Gy | 有争议 |

> **当前标准实践**：HPV+ 不降剂量——仍按 70 Gy 标准方案。降剂量仅限前瞻性临床研究。

## 三、诱导化疗后 GTV 退缩的处理

> 这是根治性 RT 中最容易出错的情景——也是与 PORT 完全不同的挑战。

### 3.1 核心困境

```
诱导化疗前 MRI：GTV = 4×3 cm（原始肿瘤范围）
诱导化疗后定位 CT：肿瘤退缩至 1.5×1 cm
问题：CTV 外放从哪个 GTV 出发？
```

### 3.2 推荐方案（九院实践）

| CTV 层级 | 从哪个 GTV 外放 | 理由 |
|---------|---------------|------|
| **CTV-HR** | **化疗后残留 GTV** + 5-10mm | 残留灶是放化疗后最终需要杀灭的肿瘤 |
| **CTV-IR** | **化疗前原始 GTV** + 10-15mm | 🔑 关键！退缩区可能仍有镜下残留——化疗前范围不能丢 |
| **CTV-LR** | 化疗前淋巴结区域 | 预防区按原始范围 |

```
化疗前 GTV 范围：████████████  ← CTV-IR 从这里外放
化疗后残留 GTV：    ████      ← CTV-HR 从这里外放
```

### 3.3 绝对禁忌

> ❌ **绝对不要仅从化疗后残留 GTV 外放所有 CTV 层级。**
> 化疗后的"影像学 CR" ≠ "病理学 CR"——口咽癌中影像学 CR 与病理 pCR 的一致性约 60-75%。
> 仅从退缩后 GTV 外放 = 漏掉退缩区的镜下残留 = 复发。

### 3.4 需要哪些影像

| 影像 | 用途 |
|------|------|
| **诱导化疗前 MRI** | 原始 GTV 范围——确定 CTV-IR/LR 的基础 |
| **诱导化疗后定位 CT** | 残留 GTV + 正常组织——计划设计的基础 |
| **PET-CT（如有）** | 辅助判断残留活性 vs 坏死/纤维化 |

> 理想流程：化疗前 MRI → 形变配准至定位 CT → 化疗前 GTV 投影到定位 CT 上 → 化疗后残留 GTV 在定位 CT 上直接勾画 → 两者结合。

## 四、颈部淋巴结管理——根治性 RT

### 4.1 口咽癌根治性 RT 颈部覆盖

| 原发部位 | cN0 | cN+ |
|---------|-----|-----|
| **扁桃体** | 同侧 II-IV + RP | 同侧 II-IV + RP ± 对侧 II-III（T3-4或近中线） |
| **舌根** | **双侧 II-IV + RP** | 双侧 II-V + RP |
| **软腭** | **双侧 II-IV + RP** | 双侧 II-V + RP |
| **咽侧壁** | 同侧 II-IV + RP | 同侧 II-V + RP |

### 4.2 与术后 PORT 的颈部覆盖差异

| | 术后 PORT | 根治性 RT |
|---|---|---|
| **cN0 颈部** | 择区预防（Ib+II+III） | 同 PORT 但 RP 必含 |
| **RP** | 口腔癌仅后门；口咽癌全部 | **口咽癌全部必照** |
| **对侧** | 仅近中线或高危 | 舌根/软腭→双侧必需 |
| **N+ 剂量** | 手术已清扫→残留风险较低 | **GTV-N 需 70 Gy**（未被清扫过） |

## 五、OAR 约束——根治性 RT 专属

> 与术后 PORT 不同：根治性 RT 的靶区更大（含 GTV），剂量更高（70 Gy），OAR 保护压力更大。

### 5.1 硬约束（一票否决）

| OAR | 约束（35fx） | 超了→ |
|-----|------------|--------|
| 脑干 Dmax | ≤ 54 Gy | ❌ 退回 |
| 脊髓 Dmax | ≤ 45 Gy | ❌ 退回 |
| 视交叉 Dmax | ≤ 54 Gy | ❌ 退回 |
| 视神经 Dmax | ≤ 54 Gy | ❌ 退回 |
| 晶状体 Dmax | ≤ 10 Gy | ❌ 退回 |

### 5.2 软约束（口咽癌专属优先级）

| OAR | 约束 | 优先级 | 超标时 |
|-----|------|:--:|------|
| **对侧腮腺 Dmean** | ≤ 26 Gy | 🔴 最高 | 不可逆口干→牺牲靶区边缘 |
| 患侧腮腺 Dmean | ≤ 30 Gy | 🟡 | 尽量保护 |
| 下颌骨 Dmax | ≤ 70 Gy | 🟡 | V60<30% |
| 咽缩肌 Dmean | ≤ 50 Gy | 🟡 | 吞咽困难有康复 |
| 口腔 Dmean | ≤ 40 Gy | 🟢 | |
| 喉 Dmean | ≤ 45 Gy | 🟢 | |
| 耳蜗 Dmean | ≤ 45 Gy | 🟢 | |

### 5.3 口咽癌根治性 RT 的 OAR 特殊考量

| 考量 | 说明 |
|------|------|
| **对侧腮腺 > 患侧腮腺** | 对侧不照的区域→腮腺保全好→口干减轻 50% 以上 |
| **下颌骨 V60Gy 预测 ORN** | V60Gy < 30%（常规）/ < 15%（牙列差/吸烟） |
| **咽缩肌——吞咽的代价** | 70 Gy 覆盖咽缩肌常不可避免（扁桃体/舌根紧邻）→吞咽康复计划 |
| **喉——声带保全** | 舌根癌→喉常被部分覆盖→Dmean 尽量 < 40 Gy |

## 六、同步化疗——根治性 RT 的标配

> 与术后 PORT 不同：根治性 RT 几乎总是联合同步化疗。

| 方案 | 用法 | 适用 |
|------|------|------|
| **顺铂 100 mg/m² Q3W × 3** | 金标准 | 肾功能正常，PS 0-1 |
| **顺铂 40 mg/m² 每周** | 替代方案 | 耐受性更好 |
| **西妥昔单抗** | 替代顺铂 | 顺铂禁忌，p16(+) |
| **卡铂+5-FU** | 二线 | 顺铂不耐受 |

> HPV+ → 同步顺铂仍是标准。降级方案（放疗60Gy+顺铂减量）仅限临床研究。

## 七、靶区勾画——根治性 RT 六步法

| 步 | 做什么 |
|----|--------|
| 1 | 调取诊断 MRI（+PET-CT）——确定原始 GTV 范围 |
| 2 | 若诱导化疗→化疗前 MRI 形变配准至定位 CT→化疗前 GTV 投影 |
| 3 | 勾画化疗后残留 GTV（定位 CT 上）+ GTV-N |
| 4 | CTV-HR = 残留 GTV + 5-10mm；CTV-IR = 化疗前 GTV + 10-15mm |
| 5 | PTV = CTV + 3-5mm |
| 6 | 颈部预防按第四节——舌根/软腭双侧，RP 必照 |
